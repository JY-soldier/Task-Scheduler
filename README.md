# 作業時間排程器-使用說明

1. 下載整個 launcher 資料夾
2. 前往 https://console.groq.com/keys 申請並記下 API key
3. 打開 Windows PowerShell 將 API key 設定成環境變數
    - 輸入 : `setx GROQ_API_KEY "你的Groq金鑰"`
    - 重新開啟 Windows PowerShell 並輸入 : `echo $Env:GROQ_API_KEY`  
      若回傳內容是你的 key 值，表示設定成功
4. 打開第一步下載的 launcher 資料夾並雙擊啟動 launcher.exe 打開網頁
5. 開始使用作業時間排程器 !!
